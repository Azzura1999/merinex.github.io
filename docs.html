Introduction

欢迎来到 ZeroGIT 官方文档。ZeroGIT 是一个专为追求极致静谧与高性能渲染而设计的下一代系统。本文档将指导您完成从安装到高级配置的所有步骤。

Installation

ZeroGIT 可以通过我们的 CLI 工具轻松安装。请确保您的系统已安装 Node.js v18+。

Terminal Copy
$ npm install -g @zerogit/cli
$ zerogit init my-project
# Initializing core engines...
# Done in 0.4s

安装完成后，您可以通过运行 zerogit dev 来启动本地开发环境。系统会自动检测您的硬件配置并开启光线追踪核心（如果可用）。

Core Architecture

ZeroGIT 基于独特的 "Starlight" 渲染管线。与传统框架不同，我们采用了去中心化的状态管理，使得每一个组件都能独立“发光”，如同星海中的恒星。

"Silence is not empty, it's full of answers. ZeroGIT brings the silence to your workflow."

想要深入了解底层原理，请参考 API Reference 章节中的 Engine.Core 模块。

Configuration

在项目根目录下的 zerogit.config.js 文件中进行全局配置。 

export default {
  theme: 'starlight',
  performance: {
    rayTracing: true,
    maxFPS: 144
  },
  // Enable experimental bloom effects
  bloom: true
}
